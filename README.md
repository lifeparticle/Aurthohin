Gibberish (অর্থহীন) Bangla text generator.





1. Make aurthohin.rb executable

```
chmod +x aurthohin.rb
```

2. Run the script

```
./aurthohin.rb

# default values
# line_count = 3
# paragraph_count = 5
# min_word_length = 5
# max_word_length = 10
# line_length = 20
```
