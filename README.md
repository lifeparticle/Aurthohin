Setup
============

1. Make aurthohin.rb executable

```
chmod +x aurthohin.rb
```

Run the script

```bash
./aurthohin.rb 2 2 4 8 16
```

```bash
এইঅগপউ দঢণষ
খজএগ ঐরটডজওধ

ঢঝতল ঙণএমঢঞক
ঐভঞয় নয়ভঋঐঝয
```
